Module nlitLang

Translates text

See also:

    Lang:AddTranslation

Usage:

    local Lang = require 'nlitLang'
local text = Lang:l('my long story text','ru') -- will translate this text to Russian
-- Translations must be added to folder nlit-langs using Lang:AddTranslation() function.

Functions

Lang:AddTranslation (lang, tbl) Adds translation to language.

Functions

Lang:AddTranslation (lang, tbl)
Adds translation to language. Can be used anywhere.

Parameters:

  • lang string 2-char language code
  • tbl table Translations from English (key-value pairs)

Usage:

    Lang:AddTranslation('ru',{
     ['Save items'] = 'Сохранить предметы',
     ['Add items'] = 'Добавить предметы'
 })
