Module Buttons

Creates GUI Buttons This module provides functions to create and manage GUI buttons.

Usage:

    local Buttons = NGUI.Buttons

Functions

Create (text, clickFunc) Creates a button
Accept (text, clickFunc) Accept button green colored
Decline (text, clickFunc) Decline button red colored
AddToCMenu (title, icon, iconSize, func) Adds button to C button menu
HudSettings () Specific HUD Settings button

Create (text, clickFunc)
Creates a button

Parameters:

  • text string Button text
  • clickFunc function OnClick function

Usage:

    Buttons:Create('Accept',function(btn)
     -- OnClick function
	btn:SetColor(NC:Red())
end)
