# Nlit's Base Framework

Contains a base functions for future garry's mod addons. Free to use, free to contribute

## Contains

- Addons Configurator
- String formatters
- Pre-configured customisable GUI, icons
- Draw functions
- Database functions
- Colors
- Fonts
- HTTP requests
- Languages
- Table functions
- Gamemodes Compatibilities

And more!
